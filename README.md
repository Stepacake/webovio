# webovio

## Description

Practice Project: layot of a single-page website. Using a flexbox.

## Installation and Running

1st. Clone the repository:
```bash
git clone https://github.com/Stepacake/webovio.git
```

2nd. Change to the project folder:
```bash
cd webovio
```

3rd. Run the Project (Google Chrome):
```bash
google-chrome index.html
```
